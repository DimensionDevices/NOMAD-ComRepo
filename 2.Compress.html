NOMAD File Compressor

Compress and decompress .txt, .md, and .com files the same way the NOMAD file manager does — gzip, then base64. Runs entirely in your browser; nothing is uploaded anywhere.

⚠️ Your browser doesn't support CompressionStream / DecompressionStream, so compression and decompression won't work here. Try a recent Chrome, Edge, or Firefox.

Base64 payload preview (what gets stored on disk)
The downloaded file is ready to drop straight into the NOMAD file manager's upload, or to overwrite the file already on the device's storage — it contains exactly what the device would store after compression. If the compressed size ends up larger than the original (common for very short files, since gzip has fixed overhead), the file manager would skip compression and store it raw — this tool always shows you both numbers so you can decide.

If the file you load isn't actually gzip-compressed (just plain text), this tool detects that automatically and hands it back unchanged — exactly what autoDecodeFileContent() does on the device's file manager.
Everything above runs locally via the browser's native CompressionStream / DecompressionStream APIs. No network requests are made.
